# Data Directory

This directory contains data manifests, sample metadata, and reference file descriptors.

**Raw sequencing data and large reference files are NOT stored in this repository.** Only metadata, manifests, and configuration files are version-controlled here.

---

## Directory Structure

```
data/
├── README.md                        # This file
├── metadata/                        # Sample information and phenotypes
│   ├── sample_manifest.txt          # List of samples with BAM paths and metadata
│   └── phenotypes.csv               # Phenotypic data (heat tolerance, tick resistance, etc.)
└── reference/                       # Reference file descriptors (not the files themselves)
    ├── reference_files.md           # List of required reference files and how to obtain them
    └── chip_positions/              # BovineHD chip position files
```

---

## Sample Manifest Format

The sample manifest (`metadata/sample_manifest.txt`) lists all samples for analysis. Expected columns:

| Column | Description |
|---|---|
| SAMPLE_ID | Unique sample identifier |
| BREED | Cattle breed (e.g., Nellore, Gir) |
| BAM_PATH | Absolute path to the sorted BAM file |
| BAM_INDEX | Absolute path to the BAM index (.bai) |
| SEQUENCING | Sequencing technology (HiFi, ONT) |
| COVERAGE | Approximate genome coverage |

Example:

```
SAMPLE_ID   BREED    BAM_PATH                          BAM_INDEX
NEL001      Nellore  /data/bams/NEL001_sorted.bam      /data/bams/NEL001_sorted.bam.bai
NEL002      Nellore  /data/bams/NEL002_sorted.bam      /data/bams/NEL002_sorted.bam.bai
```

## Required Reference Files

For information on how to obtain and set up reference data, see [docs/INSTALLATION.md](../docs/INSTALLATION.md) or run:

```bash
bash pipelines/SV_Catalog/src/utils/setup_reference_data.sh
```

Key reference files needed:

| File | Source | Size | Download |
|---|---|---|---|
| ARS-UCD2.0.fa | NCBI GCF_002263795.3 | 3.1 GB | Automated |
| ARS-UCD2.0.fa.fai | Generated by samtools | Small | Automated |
| bovhd_arsucd2.bed | Illumina / institution | Small | Manual |
| zebu_snpmap_kasarapu2017.txt | Kasarapu et al. 2017 | Small | Manual |
| VEP cache (bos_taurus) | Ensembl | ~15 GB | Automated |

## Data Policy

- Do not commit raw sequencing data (FASTQ, BAM, CRAM) to this repository
- Do not commit large reference files (FASTA, VEP cache)
- Sample metadata may contain sensitive information — review before sharing publicly
- All data used in COWADAPT analyses must comply with data access agreements from original data providers
